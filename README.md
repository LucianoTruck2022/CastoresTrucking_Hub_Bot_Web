# 🦫 CastoresHub

> Plataforma tecnológica oficial de **Castores Trucking**.

CastoresHub es el ecosistema central de Castores Trucking, diseñado para reemplazar completamente DriverHub Legacy y unificar la administración de conductores, Staff, recursos, eventos y futuras herramientas dentro de una arquitectura moderna, modular y escalable.
